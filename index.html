Services

Siya Driver Training is a leading driving school in Durban. We offer driving lessons from code 08 to code 14, and after licence courses in Durban.

Code 08 K53 Driving Lessons

Allows you to drive a vehicle less than 3500kg with a trailer less than 750kg.

Code 10 K53 Driving Lessons

Allows you to drive a vehicle over 1600kg. The trailer must not be over 750kg.

Code EC(14) K53 Driving Lessons

Allows you to drive a vehicle over 1600kg, that has trailer over 750kg. Code EC also legally allows you to drive code B(08), Code EC1(10) and Code C(11) vehicle with or without a trailer.
